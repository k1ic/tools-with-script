# douyin 

使用python3为图片添加抖音效果，生成动态的抖音效果图。

## 依赖

- pillow
- numpy
- imageio

可以使用 `pip3` 进行安装：

```sh
pip3 install pillow numpy imageio
```

## 使用

将 [douyin.py](douyin.py) 中的 **PIC_PATH** 改成相应图片路径即可执行脚本：

```sh
python3 douyin.py
```
